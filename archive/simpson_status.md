## 10. Claude Code 리모트 끊김 이슈

- **원인**: 클로드 코드 공식 버그 (GitHub Issue #34531 확인됨)
- **증상**: 세션 끊긴 후 `/rc`로 재연결해도 싱크 복구 안 됨
- **해결책**:
  1. **tmux 사용** (권장)
     ```bash
     tmux new -s claude       # 세션 시작
     tmux attach -t claude    # 재접속
     ```
  2. **SSH keepalive** (`~/.ssh/config`에 추가)
     ```
     Host *
       ServerAliveInterval 60
       ServerAliveCountMax 3
     ```
  3. **mosh** — 네트워크 끊겨도 자동 재연결 (서버에 별도 설치 필요)
