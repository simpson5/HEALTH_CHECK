### 대시보드 (static/index.html)
- [x] **홈 탭**: 현재 체중 + 목표 달성률 + 단백질 프로그레스 바 + 오늘 식단 요약 + 최근 운동
- [x] **식단 탭**: 날짜 이동(◀▶) + 단백질 링 + 끼니별 카드 + 사진 표시
- [x] **체중 탭**: 일별 체중 그래프 (7일 이동평균 + 인바디 마커) + 체성분 그래프 (체지방률/근육vs지방/BMI/기초대사) + 인바디 선택형 표시
- [x] **운동 탭**: 주간 운동 현황 (0/4회) + 운동 시작 버튼 + 날짜별 기록 + 카테고리별 가이드 (머신/맨몸/유산소)
- [x] **달력 탭**: 월별 달력 + 이벤트 도트 + 다가오는 일정
- [x] **기록 탭**: 주간 탐색(◀▶) + 서브 탭(일일/주간분석/투약/약물농도) + 접기/펼치기
