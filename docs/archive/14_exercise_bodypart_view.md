# 운동 탭 부위별 보기 계획서

> 작성일: 2026-03-25
> 목적: 머신 운동을 카테고리(머신/맨몸/유산소) 외에 부위별로도 볼 수 있게

---

## 1. 현재 구조

```
운동 가이드
[머신 7종] [맨몸 3종] [유산소 3종]

머신 탭:
  머신 체스트 프레스
  랫 풀다운
  머신 숄더 프레스
  레그 익스텐션
  라잉 레그 컬
  머신 시티드 로우
  플레이트 로우
```

문제: 머신 7종이 순서대로 나열. 부위별 구분 없음.

---

## 2. 부위 분류

| 부위 | 운동 |
|------|------|
| **가슴** | 체스트 프레스 |
| **어깨** | 숄더 프레스 |
| **등** | 랫 풀다운, 시티드 로우, 플레이트 로우 |
| **하체 앞** | 레그 익스텐션 |
| **하체 뒤** | 라잉 레그 컬 |

더 단순하게 묶으면:

| 그룹 | 운동 | 수 |
|------|------|---|
| **상체 밀기** (가슴/어깨) | 체스트 프레스, 숄더 프레스 | 2 |
| **상체 당기기** (등) | 랫 풀다운, 시티드 로우, 플레이트 로우 | 3 |
| **하체** | 레그 익스텐션, 라잉 레그 컬 | 2 |

---

## 3. UI 방안

### 방안 A: 서브탭 추가

```
[머신] [맨몸] [유산소]
  ↓
[전체] [상체밀기] [상체당기기] [하체]

상체 밀기:
  🏋️ 체스트 프레스  가슴, 어깨
  🏋️ 숄더 프레스    어깨, 삼두
```

- 탭이 2단이라 복잡할 수 있음

### 방안 B: 부위 헤더로 그룹핑 (추천)

```
[머신] [맨몸] [유산소]

── 상체 밀기 (가슴/어깨) ──
  🏋️ 체스트 프레스
  🏋️ 숄더 프레스

── 상체 당기기 (등) ──
  🏋️ 랫 풀다운
  🏋️ 시티드 로우
  🏋️ 플레이트 로우

── 하체 ──
  🏋️ 레그 익스텐션
  🏋️ 라잉 레그 컬
```

- 탭 1단 유지, 부위별 섹션 헤더로 구분
- 한눈에 어떤 부위를 훈련하는지 파악 가능

### 방안 C: 부위 탭으로 대체

```
[전체] [가슴] [등] [어깨] [하체] [맨몸] [유산소]
```

- 머신/맨몸 구분 없이 부위 기준
- 운동 추가되면 유연하게 대응

---

## 4. 추천: 방안 B

이유:
- 현재 탭 구조(머신/맨몸/유산소) 유지
- 머신 탭 안에서 부위별 섹션 헤더만 추가
- 코드 변경 최소
- 운동 추가돼도 부위만 태그하면 자동 분류

---

## 5. 데이터 변경

exercise_library에 `bodypart` 필드 추가:

| id | bodypart |
|----|----------|
| chest_press | push (상체 밀기) |
| shoulder_press | push |
| lat_pulldown | pull (상체 당기기) |
| seated_row | pull |
| plate_row | pull |
| leg_extension | legs (하체) |
| leg_curl | legs |

---

## 6. 운동 세션 페이지에도 적용

운동 시작 → 머신 탭에서도 동일하게 부위별 그룹핑:

```
── 상체 밀기 ──
[체스트프레스] [숄더프레스]

── 상체 당기기 ──
[랫풀다운] [시티드로우] [플레이트로우]

── 하체 ──
[레그익스텐션] [레그컬]
```
