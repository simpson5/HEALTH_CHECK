# 운동 기록 웹앱 기획서 v2

> 작성일: 2026-03-18 (v2 확정)
> 목적: 플릭 AI 앱 완전 대체. 웹에서 운동 기록 전부 처리.

---

## 1. 핵심 컨셉

- **루틴/분할 없음** — 전신 머신 6종, 순서 자유 선택
- **세트마다 실시간 입력** — 1세트 끝 → KG/회 입력 → 완료 → 다음 세트
- **이전 기록 자동 로드** — 두 번째부터 지난번 KG/회 미리 채워줌
- **유산소 별도 기록** — 트레드밀 시간/경사 등
- **자동 저장** — 운동 완료 시 JSON + git commit
- **대시보드 즉시 반영**

| 2 | 랫 풀다운 | 등, 이두 |

| 6 | 머신 시티드 로우 | 등, 이두 |

### 유산소
| # | 운동명 | 기록 항목 |
|---|--------|----------|
| 1 | 경사 트레드밀 | 시간(분), 경사(%), 속도(km/h) |

## 3. 화면 흐름

```
운동 탭 [운동 시작]
  → 운동 선택 화면 (6종 + 유산소)
    → 운동 골라서 탭
      → 세트 기록 화면 (KG/회 입력, 세트 추가)
        → 다른 운동 선택 or 완료
          → 운동 완료 화면 (요약)
            → JSON 저장 + git commit
```

---

## 4. 화면 설계

### 4-1. 운동 선택 화면

```
┌─────────────────────────┐
│  ← 운동 선택             │
│                         │
│ ── 근력 ──              │
│ [머신 체스트 프레스]  ✓   │ ← 완료한 운동 체크 표시
│ [랫 풀다운]              │
│ [머신 숄더 프레스]        │
│ [레그 익스텐션]           │
│ [라잉 레그 컬]            │
│ [머신 시티드 로우]        │
│                         │
│ ── 유산소 ──            │
│ [경사 트레드밀]           │
│                         │
│        [운동 완료]        │ ← 전체 종료
└─────────────────────────┘
```

- 운동 탭하면 → 세트 기록 화면으로 이동
- 이미 기록한 운동은 ✓ 표시 + 볼륨 요약
- 순서 자유 — 아무거나 골라서 진행

### 4-2. 세트 기록 화면 (근력)

```
┌─────────────────────────┐
│ ← 머신 체스트 프레스      │
│   가슴, 어깨              │
│                         │
│ 지난 기록: 40kg × 8회 × 3세트 │ ← 이전 기록 참고
│                         │
│  #   KG     회    완료   │
│ ─── ────  ────  ────    │
│  1  [40]   [8]   [ ]    │ ← 미리 채워짐 (이전 기록)
│  2  [40]   [8]   [ ]    │
│  3  [40]   [8]   [ ]    │
│                         │
│ [+ 세트 추가]            │
│                         │
│ [← 운동 목록]            │
└─────────────────────────┘
```

- 첫 사용: KG/회 빈칸 → 직접 입력
- 두 번째부터: 지난번 값 자동 채움
- 세트 완료 탭하면 ✓ 체크 (배경색 변경)
- KG/회 변경 가능 (세트마다 다를 수 있음)
- 세트 추가 버튼으로 무제한 추가
- 숫자 입력 시 **모바일 숫자 키패드** 자동 표시 (`inputmode="numeric"`)

### 4-3. 유산소 기록 화면

```
┌─────────────────────────┐
│ ← 경사 트레드밀           │
│                         │
│ 시간:  [30] 분           │
│ 경사:  [12] %            │
│ 속도:  [5.5] km/h        │
│                         │
│ 지난 기록: 30분 경사12%   │
│                         │
│ [완료]                   │
│                         │
│ [← 운동 목록]            │
└─────────────────────────┘
```

### 4-4. 운동 완료 화면

```
┌─────────────────────────┐
│                         │
│     💪 운동 완료!        │
│     소요: 75분           │
│                         │
│ 체스트프레스  40kg 3세트  │
│ 랫풀다운    35kg 3세트   │
│ 숄더프레스   25kg 3세트  │
│ 트레드밀    30분 경사12% │
│                         │
│ 총 볼륨: 3,600kg        │
│ 추정 소모: ~450kcal     │
│                         │
│ [저장하기]               │
└─────────────────────────┘
```

## 6. "이전 기록 자동 로드" 로직

```
1. 운동 선택 시 → exercise_records에서 해당 운동의 최근 기록 찾기
2. 있으면 → KG/회/세트 수를 폼에 미리 채움
3. 없으면 → 빈 폼 (3세트 기본 제공)
4. 사용자가 수정하거나 그대로 사용
```

## 7. API
