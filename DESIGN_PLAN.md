# 대시보드 디자인 전면 개선 계획서

> 작성일: 2026-03-24
> 목적: 현재 대시보드 전체 디자인 품질 향상

---

## 1. 현재 문제점 분석

### 1-1. 전체적 문제

| 문제 | 현재 | 영향 |
|------|------|------|
| 디자인 시스템 없음 | 카드/버튼/텍스트가 제각각 | 통일감 없음 |
| 카드 디자인 단조 | 전부 #13131c 배경 + #22223a 테두리 | 계층 구분 안 됨 |
| 여백 불규칙 | padding 14~20px 혼재 | 답답하거나 붕 뜸 |
| 폰트 크기 혼재 | 10~16px 사이 무작위 | 위계 불명확 |
| 색상 단조 | accent(파랑)만 과다 사용 | 중요한 게 안 눈에 띔 |
| 하단 네비 밋밋 | 텍스트+이모지만 | 앱 느낌 안 남 |
| 상단 탭 평범 | 밑줄만 있는 텍스트 탭 | 세련되지 않음 |
| 차트 기본형 | Chart.js 기본 스타일 | 커스텀 부족 |

### 1-2. 탭별 문제

| 탭 | 문제 |
|----|------|
| 홈 | 카드가 많아서 스크롤 길음. 정보 위계 불명확 |
| 식단 | 날짜 네비 + 단백질 링 + 칩 + 카드 = 스타일 혼재 |
| 체중 | 차트 + 인바디 카드 연결이 어색 |
| 운동 | 주간 현황 + 시작 버튼 + 날짜 + 가이드 = 너무 많은 요소 |
| 달력 | 도트 너무 작음, 달성률 색상 미묘 |
| 기록 | 서브 탭 안의 카드가 밋밋 |

## 2. 디자인 시스템 정의

### 2-1. 색상 팔레트

```
배경:
  --bg-primary: #0a0a0f        (메인 배경)
  --bg-card: #12121a           (카드 배경)
  --bg-card-hover: #1a1a26     (카드 호버/서브)
  --bg-elevated: #1e1e2a       (강조 카드)

텍스트:
  --text-primary: #f0f0ff      (메인 텍스트)
  --text-secondary: #8888aa    (보조 텍스트)
  --text-muted: #555570        (비활성)

액센트:
  --color-primary: #00e5ff     (메인 액션)
  --color-success: #00ff88     (달성/긍정)
  --color-warning: #ffaa00     (주의/진행)
  --color-danger: #ff4466      (경고/부정)
  --color-info: #b47fff        (정보/보조)

그라데이션:
  --gradient-primary: linear-gradient(135deg, #00e5ff, #00ff88)
  --gradient-warm: linear-gradient(135deg, #ff8c00, #ffcc00)
  --gradient-danger: linear-gradient(135deg, #ff4466, #ff8c00)
```

### 2-2. 타이포그래피

```
숫자 (체중, 수치):
  font-family: 'Bebas Neue'
  큰 수치: 36px
  중간 수치: 24px
  작은 수치: 18px

텍스트:
  font-family: 'Noto Sans KR'
  제목: 15px, 700
  본문: 13px, 400
  캡션: 11px, 400
  라벨: 10px, 700, letter-spacing: 1px
```

### 2-3. 간격 (Spacing)

```
페이지 패딩: 16px
카드 패딩: 16px
카드 간격: 12px
섹션 간격: 24px
요소 내부 간격: 8px
```

### 2-4. 카드 스타일

```
기본 카드:
  background: var(--bg-card)
  border: 1px solid rgba(255,255,255,0.06)
  border-radius: 16px
  padding: 16px

강조 카드 (중요 정보):
  background: var(--bg-elevated)
  border: 1px solid rgba(0,229,255,0.15)
  border-radius: 16px

액션 카드 (탭 가능):
  위와 동일 + cursor: pointer
  :active → scale(0.98) + 밝아짐

글라스 카드 (특별 섹션):
  background: rgba(255,255,255,0.03)
  backdrop-filter: blur(10px)
  border: 1px solid rgba(255,255,255,0.08)
```

### 2-5. 버튼

```
Primary (운동 시작 등):
  background: gradient-primary
  border-radius: 14px
  padding: 16px
  font-weight: 900
  box-shadow: 0 4px 20px rgba(0,229,255,0.2)

Secondary:
  background: var(--bg-card-hover)
  border: 1px solid rgba(255,255,255,0.1)

Tab (활성):
  background: rgba(0,229,255,0.1)
  border: 1px solid rgba(0,229,255,0.2)
  border-radius: 20px

Tab (비활성):
  background: transparent
  color: var(--text-muted)
```

## 3. 컴포넌트 재설계

### 3-1. 상단 탭 네비게이션

현재: 밑줄 있는 텍스트 버튼
변경: pill 형태 탭, 활성 탭에 그라데이션 밑줄

```
┌──────────────────────────────────┐
│ 홈   식단   체중   운동   달력   기록 │
│ ━━                               │  ← 그라데이션 밑줄 (활성만)
└──────────────────────────────────┘
```

### 3-2. 하단 네비게이션

현재: 이모지 + 텍스트 (밋밋)
변경: 아이콘 강화, 활성 탭 하이라이트

```
┌──────────────────────────────┐
│  📊        📖       🍽️       │
│ 대시보드   가이드   음식관리   │
│  ━━━                         │  ← 활성 탭 accent 도트
└──────────────────────────────┘
```

### 3-3. 날짜 네비게이션 (식단/운동/기록 공통)

현재: 둥근 버튼 + 텍스트
변경: 좌우 스와이프 제스처 + 세련된 버튼

```
     ◀  3월 24일 (화) 오늘  ▶
```

### 3-4. 프로그레스 바

현재: 단색 가는 바
변경: 둥근 모서리 + 그라데이션 + 글로우 효과

```
━━━━━━━━━━░░░░░░░░░
↑ 그라데이션 + 약한 glow shadow
```

## 4. 탭별 재설계

### 4-1. 홈 탭

```
┌─────────────────────────┐
│                         │
│  109.0 kg               │  ← 큰 숫자 중앙
│  ▼0.5kg 전일대비         │
│                         │
│  ━━━━━━▓░░░░░ 12.4%     │  ← 목표 프로그레스
│  ▼4.1kg 감량   29kg 남음 │
│                         │
├─────────────────────────┤
│ 오늘 미션         🔥 2일 │
│ ● 오전 운동  ● 저녁 운동 │  ← 2×2 그리드
│ ● 단백질     ○ 칼로리    │
│ ━━━━━━━━░░ 3/4          │
├─────────────────────────┤
│ 오늘 단백질    65g/110g  │
│ ━━━━━━━▓░░░░░░          │
├─────────────────────────┤
│ 오늘 식단                │
│ 🌅 닥터유PRO   ☀️ 차돌순두부 │  ← 가로 스크롤 칩
├─────────────────────────┤
│ 최근 운동                │
│ 오늘 07:45 랫풀+체프+트밀 │
└─────────────────────────┘
```

핵심:
- 체중 카드를 가장 임팩트있게 (큰 숫자)
- 미션 체크리스트를 2×2 그리드로 컴팩트하게
- 식단은 가로 스크롤 칩으로 간결하게

### 4-2. 식단 탭

- 날짜 네비 개선
- 단백질 링 크기 줄이고 옆에 칼로리/탄수/지방 같이 표시
- 끼니 카드에 좌측 색상 바 (아침=노랑, 점심=주황, 저녁=보라)

### 4-3. 체중 탭

- 차트 배경/그리드 더 세련되게
- 인바디 카드 그리드 디자인 개선
- 체성분 탭 디자인 통일

### 4-4. 운동 탭

- 주간 현황 카드 → 원형 프로그레스 (0/4)
- 운동 시작 버튼 → 그라데이션 + glow
- 가이드 카드 → 아이콘 + 색상 바 추가

### 4-5. 달력 탭

- 달성률 색상 더 명확하게
- 선택한 날짜 하이라이트 강화
- 이벤트 카드 디자인 개선

### 4-6. 기록 탭

- 서브 탭 디자인 개선
- 리포트 카드에 좌측 점수 색상 바

## 6. 참고 디자인 방향

다크 테마 헬스/건강 앱 기준:
- **깔끔한 카드 계층** (배경색 차이로 구분)
- **그라데이션 활용** (프로그레스 바, 주요 버튼)
- **적절한 glow 효과** (중요 수치에 은은한 빛)
- **둥근 모서리** (16px 통일)
- **여백 넉넉하게** (답답하지 않게)
- **숫자 임팩트** (큰 수치는 크게, 라벨은 작게)
